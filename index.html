Flexbox Grid

Flexbox Grid is yet another grid system for modern browsers.
It is based on the flex display property.

Check Browser Support

The motivation is that flexbox is the first specification to land in browsers that was made specifically to address problems faced in modern web design.

Features

  • Fluid
  • Simple syntax
  • Auto width
  • Nested grids
  • Alignment
  • Distribution
  • Reordering
  • Reversing

Fluid

Percent based widths allow fluid resizing of columns and rows.

.half,
.column-6 {
  flex: 0 0 50%;
}

Simple Syntax

All you need to remember is row, column, content.

<div class="row">
    <div class="column-12">
        <div class="box">12</div>
    </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="row">
    <div class="auto">
        <div class="box">auto</div>
    </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="row">
    <div class="auto">
        <div class="box">auto</div>
            <div class="row">
                <div class="auto">
                    <div class="box">auto</div>
                </div>
            </div>
        </div>
    </div>
</div>
