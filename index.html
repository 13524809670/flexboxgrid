Flexbox Grid

Flexbox Grid is yet another grid system for modern browsers that support the flex display property.

Check Browser Support

Features

  • Fluid
  • Simple syntax
  • Auto width
  • Nested grids
  • Centering
  • Alignment
  • Distribution
  • Reordering
  • Reversing

Add a row. Nest numbered columns in it.

    <div class="column-12">
        <div class="box">12</div>
    </div>
</div>
